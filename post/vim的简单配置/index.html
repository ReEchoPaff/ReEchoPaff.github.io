Vim的简单配置

2019-12-10
linux
约 328 字 预计阅读 1 分钟

环境

  • vim 8.0.1453
  • ubuntu 18.04

备份

mv ~/.vim ~/.vimbak mv ~/.vimrc ~/.vimrcbak

下载vimrc配置文件

github上比较有名的几个配置方案如下：

  • https://github.com/amix/vimrc
  • https://github.com/humiaozuzu/dot-vimrc
  • https://github.com/spf13/spf13-vim

我这里采用第二种方案，以后再试试其他的方案。

1
2

    git clone git://github.com/humiaozuzu/dot-vimrc.git ~/.vim
    ln -s ~/.vim/vimrc ~/.vimrc

下载vim插件管理工具Vundle

1

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/vundle

安装插件

使用vim任意打开某一个文件（忽略出现的各种错误），命令模式下输入:

1

    :BundleInstall

此时如果报错，则在 ~/.vimrc 文件中加上：

1

    set shell=/bin/bash

以及会报错vim-matchit插件无法安装，修改插件安装文件。

1

    sudo vim ~/.vim/bundles.vim

修改配置：

1
2

    " Bundle 'edsono/vim-matchit'
    Bundle 'thisivan/vim-matchit'

重新执行:BundleInstall，此时插件就都下载成功了。

配色方案

这个配置方案的默认的配色方案是 solarized，如果不习惯，就打开使用vim打开任意一个文件，在命令行下输入:

1

    :colorscheme [空格] [tab]

通过不停的按 tab 键可以切换不同的配色方案。
